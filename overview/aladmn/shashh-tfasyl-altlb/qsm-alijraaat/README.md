# قسم الإجراءات

<p align="right">1- تعويض العميل.<br>2- إظهار أيصال جاهز للطباعة لإرساله للعميل.<br>3- إرسال إشعار للعميل للتواصل معنا عن طريق الشات عن طريق التطبيق.<br>4- إلغاء الطلب.<br>5- تعيين مندوب او تغيير المندوب في حالة التعيين اليدوي أو المانيوال.<br>6- تغيير حالة الطلب سواء Delivered او OnWay علي حسب مرحلة الطلب و بناء علي طلب المندوب.</p>
